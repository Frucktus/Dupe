# Dupe-SO2
Скрипт для дюпа Серебра Standoff 2 [No-Ban]

1.Для Termux [No Root]:

   apt update                                                                                                                                                       
   apt upgrade                                                                                                                                                       
   pkg install python                                                                                                                                               
   apt install git                                                                                                                                                   
   git clone https://github.com/Frucktus/Dupe                                                                                                                       
   cd Dupe
   cd Dupe-Silver
   python Dupe-NoPIP.py                                                                                                                                             

2.Для Termux [Root]:

   apt update                                                                                                                                                       
   apt upgrade                                                                                                                                                       
   pkg install python                                                                                                                                               
   apt install git                                                                                                                                                   
   pip install colorama                                                                                                                                             
   git clone https://github.com/Frucktus/Dupe                                                                                                                       
   cd Dupe
   cd Dupe-Silver
   python Dupe-Silver.py


2.Для Linux:

   apt update                                                                                                                                                       
   apt upgrade                                                                                                                                                       
   sudo apt-get install python3                                                                                                                                               
   apt install git                                                                                                                                                   
   pip install colorama                                                                                                                                             
   git clone https://github.com/Frucktus/Dupe                                                                                                                       
   cd Dupe
   cd Dupe-Silver
   python3 Dupe-Silver.py
